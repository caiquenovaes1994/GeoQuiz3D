# Release Notes - v1.4.0 🚀

Temos o prazer de anunciar a versão **1.4.0** do Geo-Quiz 3D! Esta atualização traz um sistema de cadastro local seguro e aprimorado.

## ✨ Novidades e Funcionalidades

- **Cadastro Simplificado**: Agora novos jogadores podem criar uma conta local usando apenas E-mail e Senha. O sistema gera automaticamente um *username* único para facilitar o processo.
- **Validação de Senha Forte**: Mais segurança para as contas! As senhas exigem no mínimo 8 caracteres, contendo pelo menos uma letra maiúscula, um número e um caractere especial (validação *frontend* e *backend*).
- **Feedback de Conta Criada**: Melhoria na interface de Login que exibe uma mensagem de sucesso após o registro.
